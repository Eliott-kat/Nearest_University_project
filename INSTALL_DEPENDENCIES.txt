# Instructions d'installation des dépendances

Pour installer cette version d'AcadCheck, vous devez installer ces packages Python :

## Installation via pip :
pip install flask==3.0.0
pip install flask-sqlalchemy==3.1.1
pip install flask-login==0.6.3
pip install flask-dance==7.0.0
pip install gunicorn==21.2.0
pip install psycopg2-binary==2.9.9
pip install sqlalchemy==2.0.25
pip install werkzeug==3.0.1
pip install python-dotenv==1.0.0
pip install requests==2.31.0
pip install oauthlib==3.2.2
pip install pyjwt==2.8.0
pip install pypdf2==3.0.1
pip install python-docx==1.1.0
pip install weasyprint==61.2
pip install email-validator==2.1.0
pip install scikit-learn==1.3.2
pip install numpy==1.26.2

## Ou installation en une ligne :
pip install flask flask-sqlalchemy flask-login flask-dance gunicorn psycopg2-binary sqlalchemy werkzeug python-dotenv requests oauthlib pyjwt pypdf2 python-docx weasyprint email-validator scikit-learn numpy

## Démarrage :
python run_local.py

L'application sera accessible sur http://localhost:5000
